teltgram接收微信消息

使用EH Forwarder Bot实现Telegram同时收发多个微信/QQ消息

收发微信

项目地址

Telegram模块

https://github.com/blueset/efb-telegram-master

微信模块

https://github.com/blueset/efb-wechat-slave

1、 安装依赖

CentOS系统

yum install file-devel libwebp-tools git screen -y

Debian/Ubuntu系统

apt install libwebp-dev libmagic-dev git screen -y

2、安装Python3.6

CentOS系统

wget https://www.moerats.com/usr/shell/Python3/CentOS_Python3.6.sh && sh CentOS_Python3.6.sh

Debian系统

wget https://www.moerats.com/usr/shell/Python3/Debian_Python3.6.sh && sh Debian_Python3.6.sh

Ubuntu系统

apt update
apt install python3-pip python3-setuptools python3-dev -y

3、安装ffmpeg

wget https://www.moerats.com/usr/down/ffmpeg/ffmpeg-git-$(getconf LONG_BIT)bit-static.tar.xz #下载ffmpeg二进制
tar xvf ffmpeg-git-*-static.tar.xz #解压文件
mv ffmpeg-git-\*/ffmpeg  ffmpeg-git-\*/ffprobe /usr/bin/ #移动ffmpeg可执行文件
rm -rf ffmpeg-git-\* #删除文件

4、安装框架

安装稳定版

pip3 install ehforwarderbot

安装开发版，建议安装开发版，bug修复快些，功能也新

pip3 install git+https://github.com/blueset/ehforwarderbot.git

5、安装TG和微信模块

 pip3 install efb-telegram-master efb-wechat-slave

6、启用模块

先新建配置文件夹和配置文件config.yaml，使用命令：

default为配置文件默认的文件夹，你也可以命名为其它的，不会玩的就默认

mkdir -p ~/.ehforwarderbot/profiles/default
vim ~/.ehforwarderbot/profiles/default/config.yaml

比如我要同时登录并收发3个微信号

master_channel: blueset.telegram
slave_channels:
- blueset.wechat
- blueset.wechat#moe123
- blueset.wechat#rats321

7、建立TG配置文件

同样的也建在default文件夹，如果你上面更改了default文件夹，那这里也要更改

mkdir ~/.ehforwarderbot/profiles/default/blueset.telegram
vim ~/.ehforwarderbot/profiles/default/blueset.telegram/config.yaml

并填写如下代码

token: "12345:moerats" 
admins:
- 765432

8、启动

ehforwarderbo #用手机微信扫描
control+c 
screen -dmS EHF ehforwarderbot #后台运行
